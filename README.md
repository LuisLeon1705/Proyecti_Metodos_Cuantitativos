# Calculadora Estrategica

Calculadora Estrategica Pro es una aplicacion web diseñada para la resolucion de problemas de optimizacion mediante modelos matematicos avanzados e integracion con Inteligencia Artificial. La herramienta permite optimizar infraestructuras de servidores, redes de latencia y presupuestos de marketing.

## Caracteristicas Principales

### 1. Optimizacion de Servidores
Implementa el algoritmo de la mochila (Knapsack Problem) mediante programacion dinamica para maximizar el valor de los microservicios desplegados respetando la capacidad maxima de hardware.

### 2. Ruta Minima de Latencia
Calcula el camino mas corto entre nodos de infraestructura utilizando programacion dinamica backward. 
- Editor visual interactivo (drag-and-drop) para crear y modificar grafos.
- Sincronizacion en tiempo real con formato JSON.
- Resaltado visual de la ruta optima calculada.

### 3. Maximizacion de ROI en Marketing
Utiliza modelos de optimizacion no lineal para distribuir presupuestos entre diferentes canales (anuncios y creadores de contenido) con el fin de maximizar el retorno de la inversion.

### 4. Consultoria con IA
Integracion con un modelo de IA que actua como CTO para analizar los resultados obtenidos y proporcionar recomendaciones estrategicas basadas en los datos.

## Tecnologias Utilizadas

- Backend: FastAPI (Python 3.9+)
- Frontend: Jinja2 Templates, Bootstrap 5
- Visualizacion de Grafos: Vis-Network
- Generacion de Reportes: FPDF
- Servidor: Uvicorn

## Instalacion

1. Clonar el repositorio:
   git clone <url-del-repositorio>

2. Instalar las dependencias:
   pip install -r requirements.txt

3. Ejecutar la aplicacion:
   python app.py

La aplicacion estara disponible en http://localhost:8000

## Estructura del Proyecto

- app.py: Punto de entrada y definicion de rutas de la API.
- programacion_dinamica.py: Logica para problemas de mochila y rutas.
- optimizacion_nolineal.py: Modelos matematicos para marketing.
- integracion_ia.py: Conexion con el servicio de analisis.
- templates/: Archivos HTML de la interfaz de usuario.

## Uso

1. Navegar al Dashboard para seleccionar una de las herramientas de optimizacion.
2. Ingresar los datos manualmente o utilizar los ejemplos precargados.
3. Ejecutar el calculo para obtener el resultado optimo.
4. Opcionalmente, solicitar un analisis a la IA y descargar el reporte en formato PDF.
